物理的仓库：

    warehouse 顶级仓库、药库/仓库
    warehouse_zone 次级/库区--这里有一个GSP分区(合格品区/待验区/不合格区)
    warehouse_container 三级仓库/货架、容器表
    storage_location 最低级仓库/货位

仓库的记录：

    PurchaseArrival 待验区
    Inventory：仓库库存表（合格）
        



order --> warehouse_zone（待验） --> warehouse_zone（合格） --> inventory正式记录储存
order --> warehouse_zone（待验） --> warehouse_zone（不合格） --> 不记录库存


也就是从 订单order 到 库存记录inventory 要经过检验合格（quarantine）
